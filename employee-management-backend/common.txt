Oracle Client Requirement

Before running the backend, verify:

node -e "console.log(require('oracledb').versionString)"

If you get errors like:

DPI-1047: Cannot locate a 64-bit Oracle Client library

then install Oracle Instant Client and configure:

PATH=C:\oracle\instantclient_23_8;%PATH%

(or corresponding Linux path).

One More Recommendation

Instead of manually entering EMPLOYEE_ID, use a sequence:

CREATE SEQUENCE EMPLOYEE_SEQ
START WITH 1
INCREMENT BY 1;

and in the repository:

INSERT INTO EMPLOYEE(
    EMPLOYEE_ID,
    EMPLOYEE_NAME,
    EMAIL,
    MOBILE_NUMBER,
    DEPARTMENT,
    DESIGNATION,
    SALARY,
    DATE_OF_JOINING,
    STATUS
)
VALUES(
    EMPLOYEE_SEQ.NEXTVAL,
    :employeeName,
    :email,
    :mobileNumber,
    :department,
    :designation,
    :salary,
    TO_DATE(:dateOfJoining,'YYYY-MM-DD'),
    :status
)

This avoids primary-key conflicts when multiple users add employees simultaneou
